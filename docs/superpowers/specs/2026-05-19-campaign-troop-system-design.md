## 1. 兵种体系

将兵种分为**基础兵种**和**特殊兵种**两个层级，共 12 种。

### 1.1 基础兵种（7 种）

提供战力，满足阵型编队条件。

| 兵种 | 定位 | 说明 | 状态 |
|------|------|------|------|
| 农民 | 采集者 | 最弱但能采资源 | 已有 |
| 士兵 | 近战均衡 | 攻防平衡 | 已有 |
| 骑士 | 重甲突击 | 高护甲 + 冲锋 | 已有 |
| 弓箭手 | 远程输出 | 远程攻击 | 已有 |
| 枪兵 | 反骑/前排 | 长矛克制骑兵，适合防线 | 新增 |
| 法师 | 范围伤害 | AOE 输出但脆皮 | 新增 |
| 骑马弓手 | 机动远程 | 高移速 + 远程，低生命 | 新增 |

### 1.2 特殊兵种（5 种）

数量少，每种绑定独特能力，占用更多编制名额。

| 特殊兵种 | 绑定能力 | 编制占用 | 解锁方式 |
|---------|---------|---------|---------|
| 军师 | 技能：战术洞察（揭示全图敌方位置 5 秒） | 2 | 第二章关卡首通 |
| 副将 | 统领独立阵型小队 | 3 | 第二章关卡首通 |
| 战鼓手 | 技能：战鼓激励（全军攻速+30% 持续 8 秒） | 2 | 第二章关卡首通 |
| 圣骑士 | 技能：神圣庇护（全军免伤 3 秒） | 3 | 第三章关卡首通 |
| 刺客 | 技能：暗杀突袭（瞬移至敌方领主旁造成大量伤害） | 2 | 第三章隐藏分支首通 |

### 1.3 特殊兵种技能规则

- 携带特殊兵种即获得其绑定技能，无技能槽上限
- 编制上限是天然约束——带更多特殊兵种意味着更少基础兵种
- 特殊兵种的招募成本高于基础兵种
- 现有 6 个主动技能保持不变，特殊兵种技能额外叠加

### 1.4 副将机制

副将是特殊兵种中唯一不绑定技能的，其能力是统领独立阵型小队：

- **分队规则**：副将从领主编队中抽调兵种，组建独立阵型小队；小队需独立满足阵型的兵种条件
- **数量限制**：同一时间最多 1 个副将生效（即领主队 + 副将队，最多两个阵型共存）
- **行为逻辑**：副将小队跟随领主，保持一定距离（后方或侧翼），维持自己的阵型排列，半自主作战
- **副将阵亡**：小队阵型解散，剩余兵种回归领主编队
- **兵力不足**：小队不满足阵型条件时阵型失效，兵种仍跟随副将
- **手动解散**：玩家可随时解散副将小队，兵种回归领主

## 2. 阵型系统

### 2.1 阵型列表（6 种）

所有阵型通过关卡首通解锁，使用时需当前编队满足兵种条件。

| 阵型 | 效果 | 使用条件 | 解锁关卡 |
|------|------|---------|---------|
| 锥形阵 | 移速+20%，正面冲击伤害+40% | ≥3 骑士 + ≥2 士兵 | 1-3 |
| 方阵 | 全军护甲+30%，移速-15% | ≥3 枪兵 + ≥2 士兵 | 1-5 |
| 弧形阵 | 远程攻击范围+25%，远程伤害+20% | ≥3 弓箭手 + ≥2 骑马弓手 | 2-2 |
| 鹤翼阵 | 两翼包抄，侧面攻击伤害+50% | ≥2 骑士 + ≥2 骑马弓手 + ≥2 士兵 | 2-4A |
| 混元阵 | 全军属性+10%，受伤时 20% 概率减伤 50% | ≥1 每种基础兵种（7种各≥1） | 3-3 |
| 天罡阵 | 法术伤害+60%，技能冷却-30% | ≥3 法师 + ≥1 军师 + ≥2 任意 | 3-4 |

### 2.2 使用规则

- 同时只能激活一个阵型，战斗中可切换（切换冷却 10 秒）
- 兵力损失导致不满足条件时，阵型自动失效
- 未解锁阵型在 UI 中显示锁定状态，标注解锁关卡
- 阵型效果叠加在领主模式（冲锋/龟缩）之上，不冲突

### 2.3 阵型站位排列

以领主面朝方向为前方：

| 阵型 | 排列方式 |
|------|---------|
| 锥形阵 | 骑士在最前方排成三角尖端，士兵在后方两侧展开，形成倒 V 型 |
| 方阵 | 枪兵组成前排横线，士兵组成后排横线，领主在中心偏后 |
| 弧形阵 | 弓箭手和骑马弓手排成弧线在后方，形成半月形包围圈 |
| 鹤翼阵 | 士兵居中，骑士和骑马弓手分别向左右两翼斜向展开 |
| 混元阵 | 近战兵种围成内圈，远程兵种围成外圈，双层同心环 |
| 天罡阵 | 法师均匀分布在领主周围成圆形，军师紧跟领主 |

### 2.4 非阵型兵种站位

激活阵型时，不属于阵型要求的多余单位按以下规则排列：

- **近战类**（农民、士兵、骑士、枪兵）→ 填充在阵型外围，优先补阵型薄弱方向
- **远程类**（弓箭手、骑马弓手、法师）→ 跟在阵型后方，自由攻击
- **特殊兵种** → 始终紧跟领主身侧，不参与阵型排列

### 2.5 无阵型时

未激活任何阵型时，维持现有 FollowerAI 逻辑——所有兵种以领主为中心松散环绕跟随。

## 3. 战役关卡系统

### 3.1 总体结构

- **3 个章节，共 17 关**
- **分支路线**：每章 1 个分支点，两条路线奖励不同，鼓励重玩
- **统一胜利条件**：消灭所有敌方领主
- **首通固定奖励**：解锁新兵种/阵型/特殊兵种，不可重复获取

### 3.2 关卡地图

```
第一章·边境烽烟（6关）
  1-1 初战告捷（教学关）
  1-2 资源争夺
  1-3 骑兵突袭                → 首通解锁：锥形阵
       ↓
  ┌─ 1-4A 山贼巢穴            → 首通解锁：枪兵
  └─ 1-4B 兽群侵袭            → 首通解锁：骑马弓手
       ↓（汇合）
  1-5 边境要塞                → 首通解锁：方阵

第二章·深入腹地（6关）
  2-1 密林遭遇                → 首通解锁：法师
  2-2 伏击战                  → 首通解锁：弧形阵
  2-3 副将来投                → 首通解锁：副将
       ↓
  ┌─ 2-4A 平原决战            → 首通解锁：鹤翼阵
  └─ 2-4B 河谷阻击            → 首通解锁：战鼓手
       ↓（汇合）
  2-5 攻城战                  → 首通解锁：军师

第三章·王都之战（5关）
  3-1 王都外围
  3-2 内城突破                → 首通解锁：圣骑士
  3-3 王宫之战                → 首通解锁：混元阵
       ↓
  ┌─ 3-4 正面决战（终章）      → 首通解锁：天罡阵
  └─ 3-S 暗影小径（隐藏）      → 首通解锁：刺客
```

隐藏关卡 3-S 的解锁条件：全程不损失超过 5 个单位通关 3-3。

### 3.3 分支规则

- 每章 1 个分支点，玩家选择一条路线
- 两条路线奖励不同，鼓励重玩选另一条
- 分支在下一关汇合，不影响主线推进

### 3.4 单关配置数据

每关由以下字段定义：

| 字段 | 说明 |
|------|------|
| id | 关卡 ID（如 "1-1", "2-4A"） |
| name | 关卡名称 |
| chapter | 所属章节 |
| enemies | 敌方领主列表（数量、兵种配置、使用阵型） |
| map_size | 战场尺寸 |
| resources | 地图资源分布 |
| player_start | 玩家初始兵种和数量 |
| reward | 首通奖励（兵种/阵型/特殊兵种 ID） |
| next | 后续关卡 ID 列表（分支用数组） |
| unlock_condition | 解锁条件（前置关卡 ID 或特殊条件） |

### 3.5 难度递进

| 章节 | 敌方领主数 | 敌方特点 |
|------|-----------|---------|
| 第一章 | 1 个 | 基础兵种，无阵型 |
| 第二章 | 1-2 个 | 混合兵种，使用简单阵型 |
| 第三章 | 2-3 个 | 特殊兵种 + 高级阵型，协同作战 |

### 3.6 失败处理

- 失败可无限重试，不扣任何东西
- 失败后回到关卡准备界面，可调整编队

## 4. 兵种图鉴系统

### 4.1 图鉴页面内容

每个兵种的图鉴页展示：

| 内容 | 已解锁 | 未解锁 |
|------|--------|--------|
| 立绘 | 完整显示 | 剪影 |
| 属性面板 | 攻击/防御/移速/射程等 | "???" |
| 兵种描述 | 背景故事 + 战术定位 | 仅显示解锁条件 |
| 使用统计 | 累计招募数、击杀数、阵亡数 | — |
| 当前等级 | 等级进度条 + 加成数值 | — |

### 4.2 成长系统

- **等级上限**：10 级
- **每级加成**：该兵种全属性 +2%（攻击、防御、移速等），满级 +20%
- **经验来源**：每场战斗中该兵种的击杀数累计为经验值
- **经验曲线**：逐级递增

| 等级 | 累计经验（击杀数） |
|------|------------------|
| 1→2 | 20 |
| 2→3 | 30 |
| 3→4 | 40 |
| 4→5 | 50 |
| 5→6 | 60 |
| 6→7 | 70 |
| 7→8 | 80 |
| 8→9 | 90 |
| 9→10 | 100 |
| **总计** | **540** |

- **生效范围**：所有关卡和无尽模式永久生效，叠加在基础属性上

### 4.3 图鉴总览页

- 显示收藏进度：`已解锁 8/12`
- 按类型分栏：基础兵种（7 种）| 特殊兵种（5 种）
- 每个兵种显示缩略图 + 当前等级星标
- **全图鉴解锁奖励**（12/12）：领主获得被动光环"百战之师"——全军属性额外 +5%

## 5. 编队预设

- 玩家可保存 **5 套编队预设**
- 每套记录：兵种选择、数量分配、阵型选择、副将分队配置
- 进入关卡准备界面或无尽模式时可一键加载预设
- 预设名称可自定义（如"骑兵突击流"、"法术核心流"）
- 存档在 clientCloud 中

## 6. 领主天赋树

在现有 TalentSystem 基础上扩展为三条天赋路线，用声望点数解锁。

### 6.1 天赋路线

| 路线 | 方向 | 节点 1 | 节点 2 | 节点 3 | 节点 4 | 节点 5 |
|------|------|--------|--------|--------|--------|--------|
| 统帅之道 | 阵型与编队 | 阵型加成+5% | 编制上限+1 | 阵型切换冷却-3秒 | 阵型加成+10% | 编制上限+2 |
| 战争之道 | 战斗数值 | 全军攻击+3% | 暴击率+2% | 特殊兵种编制占用-1 | 全军攻击+5% | 暴击伤害+20% |
| 经略之道 | 资源与成长 | 初始资源+10% | 兵种经验获取+10% | 无尽模式商店折扣10% | 初始资源+20% | 兵种经验获取+15% |

### 6.2 规则

- 每条路线 5 个节点，逐级解锁
- **声望点数来源**：关卡首通（大量）+ 无尽模式结算（少量）
- **可洗点重置**：免费，无限制
- 三条路线可同时点，不互斥

## 7. 无尽模式

### 7.1 基本流程

```
选择初始编队（使用编队预设或手动配置）
        ↓
  第 1 波 → 战斗 → 结算战功币 → 商店购物
  第 2 波 → 战斗 → 结算战功币 → 商店购物
  ...
  第 5 波 → 精英波（更强敌人，双倍货币）→ 商店刷新 + 新增稀有商品
  ...
  第 10 波 → Boss 波（Boss + 敌方领主）→ 商店刷新 + 传说商品
  ...
  全军覆没 → 最终结算（波数记录、声望奖励、图鉴经验）
```

### 7.2 商店系统

战功币仅当局有效，不跨局累积。商店每次展示 4 件商品，随机从对应池中抽取。

| 商品类别 | 出现时机 | 示例 |
|---------|---------|------|
| 普通 | 每波刷新 | 补充 3 个士兵、回复全军 20% 生命、临时攻击+10%（持续 3 波） |
| 稀有 | 每 5 波新增 | 临时解锁一个阵型、额外招募 1 个特殊兵种、全军护甲+20%（永久） |
| 传说 | 每 10 波新增 | 编制上限+5、所有阵型加成翻倍（持续 5 波）、领主技能冷却清零 |

玩家可跳过购物直接进入下一波。

### 7.3 难度递增

- 每波敌方兵力 +10%
- 每 5 波额外增加新兵种类型
- 第 10 波起敌方开始使用阵型
- 第 20 波起敌方出现特殊兵种
- 无上限，理论可无限推进

### 7.4 结算奖励

| 波数 | 声望奖励 | 说明 |
|------|---------|------|
| 1-9 波 | 少量 | 基础奖励 |
| 10-19 波 | 中等 | 突破 Boss 波奖励 |
| 20+ 波 | 大量 | 每 5 波额外加成 |

战斗中所有兵种的击杀数正常计入图鉴经验。

## 8. 系统整合与数据架构

### 8.1 游戏主流程

```
主菜单
  ├─ 战役模式 → 战役地图（章节/关卡选择）
  │                 ↓
  │           关卡准备界面
  │           · 查看敌方情报
  │           · 编队（手动/加载预设）
  │           · 选择阵型
  │           · 分配副将小队
  │                 ↓
  │           进入战斗 → 胜利/失败
  │
  ├─ 无尽模式 → 选择编队 → 波次战斗循环
  │
  ├─ 兵种图鉴
  │
  ├─ 领主天赋
  │
  └─ 编队管理（预设编辑）
```

### 8.2 现有模块改动

| 现有模块 | 改动程度 | 说明 |
|---------|---------|------|
| Config.lua | 扩展 | 新增兵种属性、阵型配置 |
| GameState.lua | 扩展 | 新增战役进度、解锁状态、编队状态 |
| Entities.lua | 扩展 | 新增 7 种兵种的创建逻辑 |
| FollowerAI.lua | 修改 | 新增阵型站位逻辑、副将小队跟随逻辑 |
| Combat.lua | 小改 | 新增阵型加成的属性计算 |
| SkillSystem.lua | 扩展 | 新增 4 个特殊兵种绑定技能 |
| Renderer.lua | 扩展 | 新增兵种渲染、阵型视觉效果 |
| GameUI.lua | 扩展 | 新增战役地图、编队界面、关卡准备界面 |
| LordAI.lua | 修改 | 敌方 AI 支持使用阵型 |
| TalentSystem.lua | 扩展 | 三条天赋路线 |

### 8.3 新增模块

| 新模块 | 职责 |
|-------|------|
| CampaignData.lua | 所有关卡的配置数据（敌方配置、奖励、分支关系） |
| CampaignState.lua | 战役进度管理（解锁状态、首通记录、当前章节） |
| FormationSystem.lua | 阵型激活/失效判断、站位计算、加成应用 |
| SquadSystem.lua | 副将小队管理（分兵、归队、独立 AI） |
| CodexData.lua | 图鉴配置（兵种描述、经验曲线、等级加成） |
| CodexState.lua | 图鉴进度（解锁状态、使用统计、经验/等级） |
| EndlessMode.lua | 无尽模式流程控制（波次、商店、结算） |
| ShopSystem.lua | 无尽模式商店（商品池、刷新、购买） |
| PresetManager.lua | 编队预设的保存/加载/管理 |
